# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. Instead, please report it by messaging me on GitHub at [@coltonchrane](https://github.com/coltonchrane).

I will acknowledge your report as soon as possible and provide a timeline for a fix if necessary.
